## ユーザー登録画面

### 入力項目

| 項目名 | 種別 | 必須 | 最大文字数 | 備考 |
|--------|------|------|----------|------|
| 姓 | テキスト | ○ | 50 | |
| 名 | テキスト | ○ | 50 | |
| メールアドレス | テキスト | ○ | 254 | 重複チェック必要 |
| パスワード | パスワード | ○ | 128 | 8文字以上、英数字記号含む |
| パスワード（確認） | パスワード | ○ | 128 | 上と一致すること |
| 生年月日 | 日付 | ○ | | 13歳以上であること |
| 性別 | ラジオ | | | 男性/女性/その他/回答しない |
| 利用規約に同意 | チェックボックス | ○ | | チェック必須 |
| 登録 | ボタン | | | |
| キャンセル | ボタン | | | ログイン画面に戻る |

**バリデーション**:
- パスワードは強度チェック（弱/中/強で表示）
- エラーは各入力欄の下に表示
- メールアドレスの形式チェック
- 生年月日は過去の日付のみ可

## ユーザー一覧画面

### 検索エリア

| 項目名 | 種別 | 備考 |
|--------|------|------|
| キーワード検索 | テキスト | 名前・メールで検索 |
| 検索ボタン | ボタン | |

### 一覧テーブル

表示項目:
- ユーザーID
- 氏名（姓 + 名）
- メールアドレス
- 登録日（yyyy-MM-dd HH:mm）
- ステータス（アクティブ/停止中）
- 操作ボタン（編集・削除）

**その他**:
- 1ページ20件表示
- ページネーションあり
- 新規作成ボタンは画面右上

## 共通項目（全画面）

### ヘッダー
- ロゴ画像（クリックでホームへ）
- ログイン中はユーザー名表示
- ハンバーガーメニュー

### フッター
- コピーライト表示

### その他
- ローディング中はスピナー表示
- 成功/エラーメッセージはトースト表示（画面右上、3秒で消える）

## メモ

- 全体的にレスポンシブ対応必要
- モバイルでは入力欄のフォントサイズ16px以上（ズーム防止）
- エラーメッセージは日本語でわかりやすく
- ボタンのタップ領域は44px以上確保
